SoftAuthor

SoftAuthor Assistant is an invaluable tool for any programmer wishing to establish a presence for his/her creations on the internet. With SoftAuthor, information regarding authors, companies, individual programs and licensing can be organized and standardized. This information can then be processed by SoftAuthor to create standardized documentation for distribution with the program and for posting on the internet.
