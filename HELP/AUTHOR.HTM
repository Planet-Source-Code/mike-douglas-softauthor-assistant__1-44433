Data Screen: Author

This screen, in conjuction with the Program(cont) screen, comprise the data to be gathered for an individual software product.

Command Buttons
To the right of this screen are several command buttons enabling functions for this screen.
NewStart a new entry. The contents of a new entry are determined by the defaults set with the Default command button.
DeleteThis will bring up a selection window for existing items, highlight those you wish to delete and press OK.
LoadThis will bring up a selection window for existing items, highlight the one you wish to load and press OK.
SaveThis will save the current item to the hard drive

Data Fields
Author
Name Author's first name.
Author's last name.
Email Email address that should be used to contact the author.
 
Contact
Name First name of the person that should be used to contact the company regarding this product.
Last name of the person that should be used to contact the company regarding this product.
Email Email address of the person that should be used to contact the company regarding this product.
