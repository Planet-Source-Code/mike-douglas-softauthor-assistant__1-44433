Data Screen: Program(cont)

This screen, in conjuction with the Program screen, comprise the data to be gathered for an individual software product.

Data Fields
XML File URL The URL used to access the XML DIZ Info in the future.
Icon File URL The URL used to access the icon used for the program.
Screenshot URL The URL used to access a screenshot of the program.
Order URL The URL used to point people to order the program.
Info URL The URL used to point people to get more information about the program.
Categories Categories the author feels best fits the program. Comma delimitted.
Download URL(s) Primary, Secondary and additional download URLs where customers can download the program.
Keywords Keywords to be used to describe the program, should be comma delimited.
Specific Category Specific category that best fits the application.
Descriptions 45,80,250,450 & 2000 character descriptions of the program.
