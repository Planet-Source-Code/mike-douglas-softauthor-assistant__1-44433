Sub-Directory Contents

SoftAuthor has sub-directories containing files that are grouped in these directories with specific purpose and use to the end user.

DATA

This directory is used to store ".author", ".blurb", ".company", ".permission", ".program" and ".dat" files. These files can be exchanged between users of SoftAuthor to eliminate the need to re-type common information. None of these files should be edited with any other tool than SoftAuthor, the contents are strictly formatted.

HELP

This directory stores all of the HTML help files about SoftAuthor.

OUTPUT

A most inportant directory. This is where all the documents generated from the SUBMIT data screen are created.

TEMPLATE

In the event that the end user wishes to modify the output of e-mail messages or generated documents, ".template" files are stored here. For more information on modifying template files, see the Editing Document Templates page.
