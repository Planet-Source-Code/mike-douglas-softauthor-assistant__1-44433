Data Screen: Blurbs

This screen is used for creating and editing chunks of textual information (blurbs) to be used in software documentation. Blurbs can be an easy way to organize pieces of information used with several software products.

Command Buttons
To the right of this screen are several command buttons enabling functions for this screen.
NewStart a new entry. The contents of a new entry are determined by the defaults set with the Default command button.
DeleteThis will bring up a selection window for existing items, highlight those you wish to delete and press OK.
LoadThis will bring up a selection window for existing items, highlight the one you wish to load and press OK.
SaveThis will save the current item to the hard drive
 
OrderUsed to define the order in which blurbs will be shown in generated documentation.

Data Fields
Name A descriptive tag for this blurb.
Text Textual content of blurb.
