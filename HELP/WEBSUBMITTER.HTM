Web Submitter

An additional toolbar appears with the browser when a web submission has been started. This toolbar provides the necessary means to update and organize information used by SoftAuthor when submitting software to web sites.

Command Buttons
Move backwards one position in the submission list.
Move forwards one position in the submission list.
Update submit list site to current URL for future use.
Cut current site from submission list.
Add/Edit Sticky Notes for current site.
Checkbox to mark site submission as completed, followed by current submit list item and count.
