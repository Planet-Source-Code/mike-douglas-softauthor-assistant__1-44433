Data Screen: Company

This screen, comprises the data to be gathered for an individual company.

Command Buttons
To the right of this screen are several command buttons enabling functions for this screen.
NewStart a new entry. The contents of a new entry are determined by the defaults set with the Default command button.
DeleteThis will bring up a selection window for existing items, highlight those you wish to delete and press OK.
LoadThis will bring up a selection window for existing items, highlight the one you wish to load and press OK.
SaveThis will save the current item to the hard drive
DefaultSet current data field values to be the default values for this screen when the New command button is used. This can be very helpful when data is often repeated from item to item.

Data Fields
Name Contains company name, if the author does not have a company name they should have entered their own name under this field.
Address1 First line of the mailing address.
Address2 Second line of the mailing address.
City/Town The city, town or village where the company resides should appear in this field.
State/Province The state, province or region where the company resides.
ZIP/Postal Code Zip or Postal code.
Country Country where the company resides.
Website Should contain the basic URL to access the company's web site.
About A brief description of the company.
Contact Email Email address where general inquiries should be directed.
Contact Phone Phone number where general inquiries should be directed.
Sales Email Email address where sales inquiries should be directed.
Sales Phone Phone number where sales inquiries should be directed.
Support Email Email address where technical support inquiries should be directed.
Support Phone Phone number where technical support inquiries should be directed.
Organization Membership Organizational memberships held by company.
Fax Phone number where faxes should be directed.
