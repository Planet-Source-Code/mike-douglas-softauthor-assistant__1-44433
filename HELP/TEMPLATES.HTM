Editing Document Templates

Document Templates are simply ASCII text files, editable in any text editor (such as notepad), with special "tags" that are replaced with data when the new document is generated.

The tags all begin with "<!--", have the tag name, and then end with "-->". A tag example would be <!--NAME-->, anywhere "<!--NAME-->" appeared in the template, it would be replaced with whatever value NAME held.
Example:
Contents of template file:New file (based on template):
Dear <!--NAME-->,
<!--COMPANY-->, thanks you for your recent
purchase of <!--PRODUCT-->.

Yours Truly,
<!--COMPANY_CONTACT-->
 		Dear Joe Smith,
Big Co., thanks you for your recent
purchase of Super Widgets PRO.

Yours Truly,
Fred Jones

The included template files(in the TEMPLATE diretory) and their usage are as follows:
EMAIL Used to format message for e-mail based submission.
FILE_ID_DIZ Used to generate FILE_ID.DIZ file. Please note that FILE_ID.DIZ is a strict format, leaving very little room for information. Adding tags to this template could generate errors when generating the FILE_ID.DIZ file.
HTML Used to generate INDEX.HTML file. The generated file is intended to be used as a product homepage. It is HIGHLY suggested this template be modified to reflet the look and feel of your own website.
PAD_FILE_HTM An extremely informative web page designed more for mass information than for looks. This page may be linked to for in-depth information regarding the software product.
PAD_FILE_XML Used to generate PAD_FILE.XML file. This XML file is a HIGHLY format dependant, industry standard file and as such should not be modified unless the user is extremely familiar with the content of this file..
PRESSRELEASE Used to generate PRESSRELEAE.TXT file. A textual press release for consumption of the news media.
README Used to generate README.TXT file. Intended as a first step document included in the software distribution.

The document tags available to SoftAuthor include:
ADDRESS_1 First line of company address.
ADDRESS_2 Second line of company address.
ARTICLES Text content of combined blurbs.
AUTHOR Author file used with program.
AUTHOR_EMAIL The program author's email address.
AUTHOR_F_NAME The program author's first name.
AUTHOR_L_NAME The program author's last name.
BLURBS Comma delimited list of blurbs used for program.
CATEGORIES Categories the author feels best fits the program. Comma delimitted.
CHANGE_INFO What has changed since the last release.
CITY_TOWN The city, town or village where the company resides.
COMPANY Company file used with program.
COMPANY_ABOUT A brief description of the company.
COMPANT_CONTACT First and last name of company contact.
COMPANY_NAME Contains company name, if the author does not have a company name they should have entered their own name under this field.
COMPANY_WEBSITE_URL Should contain the basic URL to access the company's web site.
CONTACT_EMAIL Email address of the person that should be used to contact the company regarding this product.
CONTACT_F_NAME First name of the person that should be used to contact the company regarding this product.
CONTACT_L_NAME Last name of the person that should be used to contact the company regarding this product.
CONTACT_PHONE Company phone number where general inquiries should be directed.
COUNTRY Country where the company resides.
DESCRIPTION_2000 2000 character descriptions of the program.
DESCRIPTION_250 250 character descriptions of the program.
DESCRIPTION_45 45 character descriptions of the program.
DESCRIPTION_450 450 character descriptions of the program.
DESCRIPTION_80 80 character descriptions of the program.
DISTRIBUTION_INCLUDES Indicates what the program distribution includes for necessary operation (JAVA VM, Drivers, Visual Basic Runtime, etc).
DISTRIBUTION_PERMISSIONS Distribution permissions given by the author to vendors and other distributors.
DOWNLOAD_TIME_14K The amount of time to download the file at 14K.
DOWNLOAD_TIME_28K The amount of time to download the file at 28K.
DOWNLOAD_TIME_56K The amount of time to download the file at 56K.
DOWNLOAD_TIME_64K The amount of time to download the file at 64K.
DOWNLOAD_TIME_128K The amount of time to download the file at 128K.
DOWNLOAD_TIME_T1 The amount of time to download the file with a T1 connection (1.544M or 1544K).
DOWNLOAD_URL_1 Primary download URL where customers can download the program.
DOWNLOAD_URL_2 Secondary download URL where customers can download the program.
DOWNLOAD_URL_3 Additional download URL where customers can download the program.
DOWNLOAD_URL_4 Additional download URL where customers can download the program.
EULA Contains the end user license agreement for the program.
EXPIRE_BASED_ON Specifies the type of expiration.
EXPIRE_COUNT The number of days or uses until program expires.
EXPIRE_DATE Date the software product will expire.
EXPIRE_DAY Day the software product will expire.
EXPIRE_MONTH Month the software product will expire.
EXPIRE_INFO Statement sentence relating expiration information.
EXPIRE_OTHER_INFO Other important information regarding expiration of the program.
EXPIRE_YEAR Year the software product will expire.
FAX_PHONE Company phone number where faxes should be directed.
FILENAME_GENERIC Program name should be the 6.3 generic name, without the version number.
FILENAME_LONG Preferred long filename that should be used for the program.
FILENAME_PREVIOUS Program name should be the 8.3 but have the previous version number for the last 2 characters.
FILENAME_VERSIONED Program name should have an 8.3 name, the last 2 characters of the first portion of the name should be the version number.
FILESIZE_BYTES The size of the distribution file in Bytes.
FILESIZE_KB The size of the distribution file in Kilobytes.
FILESIZE_MB The size of the distribution file in Megaytes.
GENERAL_EMAIL Company email address where general inquiries should be directed.
ICON_FILE_URL The URL used to access the icon used for the program.
INCLUDES_DIRECTX Y/N value indicating directx driver inclusion in distribution.
INCLUDES_JAVA Y/N value indicating JAVA virtual machine inclusion in distribution.
INCLUDES_VB Y/N value indicating Visual Basic runtimes inclusion in distribution.
INFO_URL The URL used to point people to get more information about the program.
INSTALL_SUPPORT What type of install support does the program have.
KEYWORDS Keywords to be used to describe the program, should be comma delimited.
LANGUAGE All languages that apply to the program.
ORDER_URL The URL used to point people to order the program.
ORGANIZATION_MEMBERSHIP Organizational memberships held by company.
OS_SUPPORT All OS types that apply to the program.
PERMISSIONS Permission file used for program.
PROGRAM_EXPIRES Y/N value indicating whether the program has expiration information.
PROGRAM_NAME Contains the full name of the program.
PROGRAM_NAME_BOLD Contains the full name of the program in upper case characters.
PROGRAM_TYPE The type of program(Freeware, Shareware, etc...).
PROGRAM_VERSION The version number of the product, should NOT contain the 'v.' characters.
REGISTRATION_COST_OTHER Price of product in an alternative currency, currency symbol should be included.
REGISTRATION_COST_USD The cost of the program in United States Dollars.
RELEASE_DATE Date the software product was released.
RELEASE_DAY Day the software product was released.
RELEASE_MONTH Month the software product was released.
RELEASE_STATUS The type of release the application is defined as.
RELEASE_YEAR Year the software product was released.
SALES_EMAIL Email address where sales inquiries should be directed.
SALES_PHONE Phone number where sales inquiries should be directed.
SCREENSHOT_URL The URL used to access a screenshot of the program.
SPECIFIC_CATEGORY Specific category that best fits the application.
STATE_PROVINCE The state, province or region where the company resides.
SUPPORT_EMAIL Email address where technical support inquiries should be directed.
SUPPORT_PHONE Phone number where technical support inquiries should be directed.
SYSTEM_REQUIREMENTS Minimum system requirements for the program to function properly.
TOC A numerical table of contents of blurbs for use in generating the README.TXT file.
XML_FILE_URL The URL used to access the XML DIZ Info in the future.
ZIP_POSTAL Zip or Postal code.
