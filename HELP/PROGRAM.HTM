Data Screen: Program

This screen, in conjuction with the Program(cont) screen, comprise the data to be gathered for an individual software product.

Command Buttons
To the right of this screen are several command buttons enabling functions for this screen.
NewStart a new entry. The contents of a new entry are determined by the defaults set with the Default command button.
DeleteThis will bring up a selection window for existing items, highlight those you wish to delete and press OK.
LoadThis will bring up a selection window for existing items, highlight the one you wish to load and press OK.
SaveThis will save the current item to the hard drive
DefaultSet current data field values to be the default values for this screen when the New command button is used. This can be very helpful when data is often repeated from item to item.

Data Fields
Name Contains the full name of the program
Version The version number of the product, should NOT contain the 'v.' characters.
-(Release Status) The type of release the application is defined as.
Expiration Information Check this box if the software product has expiration information (the expiration details window will then be displayed).
Expiration Count The number of days or uses until program expires.
Expiration Base Specifies the type of expiration.
Expiration Date Date the software product will expire.
Other Info Other important information regarding expiration of the program.
Company The supporting company for the software product (defined on the Company data screen).
Author The actual author of the software product (defined on the Author data screen).
Type Select the type of program here(Freeware, Shareware, etc...).
Permission Contains permission statements for vendors, distributors and end users (defined on the Permissions data screen).
Blurbs The pieces of textual information to be used in document creation for the software product (defined on the Blurbs data screen).
OS upport All OS types that apply to the program.
Change Info What has changed since the last release.
Release Date Date the software product was released.
Registration(USD) The cost of the program in United States Dollars.
(Other) Price of product in an alternative currency, currency symbol should be included.
Language All languages that apply to the program.
Includes Indicates what the program distribution includes for necessary operation (JAVA VM, Drivers, Visual Basic Runtime, etc).
Install Support What type of install support does your program have.
Requirements Minimum system requirements for the program to function properly.
Filename Versioned(8.3) Program name should have an 8.3 name, the last 2 characters of the first portion of the name should be the version number.
Filename Previous(8.3) Program name should be the 8.3 but have the previous version number for the last 2 characters.
Filename Generic(6.3) Program name should be the 6.3 generic name, without the version number.
Filename Long Preferred long filename that should be used for the program.
File Size The size of the distribution file in Bytes.
